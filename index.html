HTML Elements

Tag Name Description
<li> List Item Element Creates a list item.
<video> Video Element Embeds a media player for video playback.
<em> Emphasis Element Emphasizes text, browsers use italics.
<ol> Ordered List Item Creates a list of items in sequential order, appears numbered.
<div> Div Element Used as a container that divides an HTML document into sections and is short for "division".
<br> Line Break Element Creates a line break in text.
<img> Image Element Embeds images in documents
<h1>-<h6> Heading Elements Creates a heading ordered from the highest level to the lowest level.
<p> Paragraph Element Contains and displays a block of text.
<ul> Unordered List Element Creates a list of items in no particular order.
<body> Body Element Represents the content of an HTML document.
<span> Span Element Can be used to group text for styling purposes.
<strong> Strong Element Highlights important, serious, or ugent text.

HTML Attributes

Attribute Name Description
id= Unique ID Attribute Can be assigned to different elements in order to differentiate between them.
alt= Alternative Test Attribute Displayed if an image fails to render.
